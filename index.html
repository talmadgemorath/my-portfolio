FiveCardStud - Poker Hand Analyzer

Description: The card game poker has many variants. In this project, I developed a program that analyzes six poker hands from the game Five Card Stud. The goal is to rank the hands from the strongest to the weakest according to poker rules. The game has no “draw” phase, and ties are broken explicitly to determine a unique winner.

Technologies Used: C++, C#, Java, Python, Fortran, Julia, Go, Perl, Rust, Lisp

How to Run:

  • C++: g++ cpp/Card.cpp cpp/Hand.cpp cpp/FiveCard.cpp; ./a.out handsets/filename
  • C#: mcs csharp/Card.cs csharp/Hand.cs csharp/FiveCard.cs; mono csharp/Card.exe handsets/filename
  • Java: javac Java/Card.java Java/Hand.java Java/FiveCard.java; java Java/FiveCard handsets/filename
  • Python: python3 python/Card.py python/Hand.py python/FiveCard.py; ./a.out handsets/filename
  • Fortran: gfortran Fortran/Card.f08 Fortran/Hand.f08 Fortran/FiveCard.f08; ./a.out handsets/filename
  • Julia: julia julia/FiveCard.jl handsets/filename
  • Go: go run Go/Card.go Go/Hand.go Go/FiveCard.go handsets/filename
  • Perl: perl Perl/FiveCard.pl handsets/filename
  • Rust: cargo run ../handsets/filename
  • Lisp: sbcl --script Lisp/FiveCard.lisp handsets/filename
